A simple React JS application that demonstrates:

🔄 Client-side routing using React Router

⚡ Conditional rendering

➕ Add / ➖ Remove activities 

🎨 Basic CSS styling

🚀 Features

View a list of activities

Add activities to list

Remove activities from list

Navigate between pages without reload

Display message when no activities exist

🛠️ Built With

⚛️ React JS (Functional Components)

🧭 React Router DOM (v6)
